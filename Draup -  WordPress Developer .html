Nature Of Business : IT/Software
Designation : WordPress Developer
Tentative Job Location : Bangalore
Description :


Draup is looking for a Php/Wordpress Developer responsible for maintaining the marketingwebsite of
the company – Draup.com.
Your main duties will include from creating pages (templates using HTML & PHP) tomaintaining
Wordpress dashboard of marketing site.

Responsibilities
1. Implementing new features and functionality from scratch
2. Ensuring high performance & availability across all devices
3. Coordinating the workflow between the graphic designer, the HTML coder (yourself)
4. Communicating with external web services (third-party plugins for required functionality)
5. Maintenance of existing features

Skills and Qualifications
1. Must have experience in developing Word press applications. Must be a Professional, with
excellent communication skills.
2. Strong understanding of PHP development
3. Creating self-contained, reusable, and testable modules and components with working
knowledge of HTML5 and CSS3
4. Ability to understand CSS changes and their ramifications to ensure consistent style across
platforms and browsers
5. Ability to provide good SEO solutions for application
6. Experience using debugging tools such as Firebug and Chrome Inspector
7. Strong understanding of front-end and back-end website development

Good to have:
1. Expertise in JS development (any Single page application development) along with HTML5,CSS3
template designs
2. Knowledge of how to interact with RESTful APIs and formats (JSON, XML) and Database
Management System (MySQL etc...)
3. Good experience with Version control system (GIT)
4. Experience with monitoring the performance of the application (lighthouse etc.)
